# tree
Alpine docker image with binary tree pre installed.

You can use it like this: `docker run --rm -t -v /tmp:/data tree`

```
$ docker run --rm -t -v /tmp:/data tree
/data
├── [ 104]  core.php
├── [ 409]  global.php
├── [ 218]  global.sample.php
└── [  62]  userconfig.json

0 directories, 4 files
```
